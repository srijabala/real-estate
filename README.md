# 🏡 City View – Real Estate Website

City View is a responsive frontend real estate website built using HTML, CSS, JavaScript, and Bootstrap. The application provides a modern interface for exploring residential properties, searching listings, and connecting with real estate agents.

## 🌐 Live Demo

🔗 https://srijabala.github.io/real-estate/

## 📂 GitHub Repository

🔗 https://github.com/srijabala/real-estate

## 📖 Overview

City View is designed to deliver a clean and intuitive property browsing experience. Users can explore featured properties, search listings, view agent information, and contact the agency through a responsive interface optimized for different screen sizes.

---

## ✨ Features

- Responsive navigation with mobile-friendly menu
- Featured property listings with pricing and location details
- Property search functionality
- Interactive Buy, Sell, and Rent popup
- Favorite (wishlist) icon for properties
- Real estate agent profiles
- Contact form for customer inquiries
- Responsive design for desktop and mobile devices

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/srijabala/real-estate.git
```

Navigate to the project directory:

```bash
cd real-estate
```

Open `index.html` in your preferred web browser.
